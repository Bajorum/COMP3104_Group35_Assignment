COMP 3104 is a course on DevOps offered at George Brown College.
The course covers key DevOps principles such as continuous integration (CI), continuous deployment (CD), version control with Git, and the use of automation tools to streamline software development processes. 
Students learn to work collaboratively in teams, implement CI/CD pipelines using tools like GitHub Actions, and deploy applications using modern cloud platforms. 
The course aims to prepare students for real-world DevOps practices, focusing on improving the efficiency and quality of software delivery.
