COMP 3104 focuses on teaching students the core principles of DevOps, including continuous integration, continuous delivery, version control systems, and automation tools like GitHub Actions and Jenkins.

DevOps is a fun class and you can learn so many things.

DevOps is a methodology in the software development and IT industry.

The term was formed by combining “development” and “operations,” which provides a starting point for understanding exactly what people typically mean when they say “DevOps.”
